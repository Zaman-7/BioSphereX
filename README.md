## Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/biospherex.git
   cd biospherex
   ```

2. **Start a local server**
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Using PHP
   php -S localhost:8000
   ```

3. **Open in browser**
   Navigate to `http://localhost:8000`
